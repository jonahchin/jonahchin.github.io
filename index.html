Firework Simulator

Firework simulator is a 2D, java based project that I built myself in 2016. The simulator uses custom physics to replicate the path of fireworks as they're launched. The application allows users to customize the fireworks' launch angle, as well the speed and direction of the wind in the environment. The most difficult part of this project was producing the individual sparks that are emitted from the moving firework.

Budget++

Budget++ is a C++ based Windows application that I built as part of a six-person agile team. The goal of this project was to build something that students can use to track their spending over a certain amount of time. The application displays data in pie charts, and allows users to compare their actual spendings to their projected budget.
